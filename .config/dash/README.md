# Dash

[공식 홈페이지](https://www.gh-dash.dev/)

## 시작하기

1. [공식 홈페이지 - 시작하기](https://www.gh-dash.dev/getting-started/)를 참고하여 설치
2. 추가 패키지 설치

```shell
brew install gum
```

## 커스텀 키바인딩

### 공통

- `I`: Issue 생성
- `P`: Pull Request 생성

### Pull Request

- `E`: Pull Request 편집

### Issue

- `E`: Issue 편집
- `D`: Issue 개발
